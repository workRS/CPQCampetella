PRINT PRICE

By deselecting the Print Price button, the prices in each row of the group are hidden. It is not possible to hide the price of a single row (it can be done manually from Word or by not inserting the row on the CPQ).
Examples:

  • if the margin of the robots is differente from the standard values (30% Export), Print Price is automatically deselected
  • if the margin of products other than robot is not standard (15% Export), Print Price will remain selected (the "pumped" price will appear directly in the printed version)
  • if you need to add hours to the robot, the Print Price must be deselected manually
  • if the margin of the robots is higher than 50%, a message will open up and ask for correction
  • if the discount is greater than the margin, a message will open up and ask for correction
  • in Services the price is always visible, you can set the price as 0 and increase the price elsewhere
